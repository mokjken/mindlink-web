怎么运行MindLink？
By Ken
----------------------------------------------------------------
STEP 1: 安装Node.js
----------------------------------------------------------------
网址：https://nodejs.org/
----------------------------------------------------------------
STEP 2: 运行
----------------------------------------------------------------
1. 打开MindLink文件夹
2. 在文件资源管理器的上方路径地址栏输入cmd，此时命令提示符会在本目录下弹出
3. 按照顺序输入以下命令：

   npm install
   cd backend
   npm install
   cd ..
   npm run dev

4. Go to http://localhost:3000
